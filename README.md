## 图片上传管理平台

### 本地代码同步工具
https://github.com/lrenc/fis-receiver

```
cd fis-receiver
nohup ./main & # 默认端口8527 可自定义
```

### 代码挑食
```
//linux
DEBUG=myapp & npm start
//windows
set DEBUG=myapp & npm start
```


### 服务器启动服务
```
forever start ./bin/www
forever list
forever stopall
```
